# ai-training

このリポジトリは Obsidian と GitHub を連携させた Windows 向け演習用プロジェクトです。Markdown を Obsidian で参照しつつ、GitHub による履歴管理と共有を行います。

## 概要
- Windows での Obsidian Vault として利用する構成
- ローカルのフォルダを GitHub リポジトリとして管理
- README.md はプロジェクト概要、CLAUDE.md は設定説明、specs/ は仕様資料

## 事前準備
1. Obsidian をインストールし、ローカルに Vault を作成する
2. claude CLI を導入し、プロジェクトセットアップを依頼する
3. GitHub アカウントと空リポジトリを用意し、origin リモートを設定する
4. Obsidian でこのフォルダを開き、Markdown が正しく表示されることを確認する
